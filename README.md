**Локальный менеджер паролей без облачной синхронизации, с целью полной безопасности.**  
Все данные хранятся исключительно на вашем устройстве.

---

## 🔐 Безопасность и вход

- **Пароль приложения** – устанавливается при первом запуске (мин. 4 символа).  
  Хешируется с помощью **PBKDF2** с уникальной солью, что защищает от радужных таблиц и брутфорса.

- **Мастер-ключ шифрования** – генерируется случайно при первом запуске.  
  Хранится **в зашифрованном виде** (паролем приложения) и расшифровывается только в момент входа.  
  Используется для шифрования всех паролей аккаунтов (AES-256-GCM).

- **Кодовая фраза восстановления** – для сброса пароля приложения, если вы его забыли.  
  Тоже хешируется через PBKDF2 с отдельной солью.

- **Биометрическая разблокировка** (отпечаток / Face ID) – реализована через **Android KeyStore**.  
  При включении мастер-ключ шифруется биометрическим ключом и сохраняется в `Settings`.  
  При входе по биометрии мастер-ключ расшифровывается только после успешной аутентификации (BiometricPrompt с CryptoObject).

- **Защита от скриншотов** – системный флаг `FLAG_SECURE` (включается/выключается в настройках).

- **Автоматическая блокировка** (по таймауту) – заложена в настройках (будет реализована в следующих версиях).

## 📂 Управление аккаунтами (паролями)

- **Добавление аккаунта** – сервис, логин, пароль, URL, заметки, категория, иконка.
- **Редактирование аккаунта** – все поля можно изменить.
- **Просмотр деталей** – на отдельном экране с возможностью скопировать логин/пароль, открыть сайт, показать/скрыть пароль.
- **Удаление аккаунта** – с подтверждением.
- **Копирование пароля** – одной кнопкой (в списке и в деталях).
- **Генератор паролей** – настраиваемая длина, символы (заглавные, строчные, цифры, спецсимволы).
- **Индикатор сложности пароля** – при вводе (5 уровней).
- **Поиск по аккаунтам** – по названию, логину, заметкам, URL.
- **Фильтрация по категориям** – горизонтальные чипсы на главной.

---

## 🏷️ Категории

- **Просмотр всех категорий** – на отдельной вкладке.
- **Добавление категории** – название, иконка, цвет.
- **Редактирование категории** – название, иконка, цвет.
- **Удаление категории** – с автоматическим переносом всех аккаунтов в категорию «Все».
- **Встроенные иконки** – более 20 предустановленных (соцсети, финансы, работа, игры и т.д.)
- **Цветовая палитра** – 8 цветов для визуального различения.

---

## 🖼️ Иконки аккаунтов

- **Выбор из предустановленного набора** – через диалог.
- **Загрузка с сайта (favicon)** – автоматически по URL (используется `icon.horse`, Google S2, DuckDuckGo).
- **Выбор изображения из галереи** – обрезка в круг.
- **Сохранение кастомной иконки** – локально в файловой системе.
- **Автоматическое масштабирование и закругление** (ShapeableImageView).

---

## ⚙️ Настройки

- **Смена пароля приложения** – требует ввода текущего пароля.
- **Смена кодовой фразы восстановления**.
- **Включение/выключение биометрии** – создаёт и удаляет ключ в KeyStore.
- **Включение/выключение защиты от скриншотов** – мгновенное применение.
- **Экспорт в JSON** – зашифрованный файл с данными (требует кодовую фразу).
- **Импорт из JSON** – восстановление данных из зашифрованного бэкапа (требует кодовую фразу).
- **Удалить все данные** – полная очистка БД и сброс настроек.

## 📱 Интерфейс

- **Тёмная тема** – только тёмный фон (`#FF12121E`).
- **Нижняя навигация** – 3 вкладки: Главная, Категории, Настройки.
- **Плавающая кнопка (FAB)** – на главной для добавления аккаунта (скрыта на других вкладках).
- **Адаптивный дизайн** – поддерживает вертикальную ориентацию.

---

## 🛠️ Дополнительные возможности

- **Копирование логина / пароля** в буфер обмена.
- **Открытие URL** во внешнем браузере.
- **Подтверждение действий** (диалоги перед удалением).
- **Кэширование favicon** – в памяти (LruCache) и на диске для быстрой загрузки.
- **Обработка ошибок** – краш-логи сохраняются в файл (через `SecureVaultApplication`).
- **Локальное хранение** – все данные только на устройстве (без интернета).

---

## 🗂️ Структура данных

- **SQLite** с таблицами: `accounts`, `categories`, `settings`.
- **UUID** – уникальные идентификаторы для записей.
- **Шифрование паролей** – AES-256-GCM с мастер-ключом.
- **Хеширование пароля и кодовой фразы** – PBKDF2 с солью.
- **Миграции БД** – поддерживаются (при обновлении добавляется поле `url`).

---

## 🧪 Технологии

- **Язык:** Java (Android SDK)
- **Архитектура:** Activity/Fragment + SQLiteOpenHelper
- **База данных:** SQLite
- **Шифрование:** `javax.crypto` (AES-GCM, PBKDF2)
- **Биометрия:** AndroidX Biometric + Android KeyStore
- **Сериализация:** Gson для JSON
- **UI:** Material Design, тёмная тема
